# :hash: Tic-Tac-Toe Marvel

This project is a Tic-Tac-Toe using Marvel characters, a challenge by Celero.
Its base was generated with [Angular CLI](https://github.com/angular/angular-cli) version 11.0.7.

## Required settings

There is a required setting in order to use [Marvel API](https://developer.marvel.com). You will need to set the environment variables `apiKey`, `hash` and `timestamp` in [src\environments\environment.ts](src\environments\environment.ts) with your Marvel credentials.

## Install

To install all project dependencies, run `npm install`.

## Development server

Run `npm run start` or `ng serve` and navigate to http://localhost:4200/.

## Build

Run `npm run build` or `ng build` to build the project. The build artifacts will be stored in the `dist/` directory. Use the `--prod` flag for a production build.

## Running unit tests

Run `ng test` to execute the unit tests via [Karma](https://karma-runner.github.io).

## Running end-to-end tests

Run `ng e2e` to execute the end-to-end tests via [Protractor](http://www.protractortest.org/).
